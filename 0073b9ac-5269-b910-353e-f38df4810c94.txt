　　第253章
<br/>　　从青牛山离开后，山贼之心就处于灰色状态，点将台失效，系统商店失效，就连系统任务也没有再出现过。现在的山贼之心，程大雷只能用来看看人物信息。
<br/>　　程大雷知道，怕是要等自己重新建立山寨，系统才会被再次激活。
<br/>　　只是没想到，这次杀了鬼见愁，反而触发了隐藏任务。
<br/>　　不过，这鬼见愁究竟是谁呐，刚才乱杀乱砍，倒也没注意哪个长得像老大。
<br/>　　不过现在也顾得不太多，还是逃命要紧，程大雷和李婉儿骑着毛驴，狂奔在草原上。
<br/>　　……
<br/>　　大战过后，废城下一片狼籍，马匪和鱼龙卫都付出了极惨烈的代价，一眼看过去，遍地尸体。
<br/>　　宫符带人打扫战场，埋葬尸体，治疗伤员。
<br/>　　这时罗酒带人回来，宫符立刻问：“找到公主了么？”
<br/>　　罗酒摇摇头，骂咧咧道：“不知那混蛋跑到哪儿去了，路上我们碰到马贼，又和他们打了一场。”
<br/>　　现在这片草原上，分散着两股势力，马贼和鱼龙卫。而双方都在寻找劫走公主的程大雷。而这双方也有刻骨仇恨，遇见了就打得不死不休。
<br/>　　现在提起程大雷就咬牙切齿，恨不得将它生吞活吃。
<br/>　　“不是说公主与程大雷有恩么，他为何还要抢走公主。”
<br/>　　“强盗怎么会有人性，他们就是一帮忘恩负义的畜生。”有人骂道。
<br/>　　今日一场大战，鱼龙卫折了很多弟兄，大家提起强盗就一肚子火，而程大雷虽然不是和鬼见愁一伙的，但同样是强盗，甚至更加可恶。
<br/>　　更关键的是，丢了明玉公主，这些人都是砍头的死罪。所以这些人必须找到程大雷，救回公主。
<br/>　　“宫符，大家现在怎么办？”罗酒问道。
<br/>　　宫符沉思片刻，道：“程大雷劫走公主，必然是往关内走，肖将军，你带人回到夕照关，传令各个关口封锁道路，许出不许进。”
<br/>　　“人都回去，怎么去救公主？”罗酒问道。
<br/>　　“程大雷只有一个人，留太多人在此地也无用，想要救回公主，需要的是龙卫。”宫符昂起头：“回京请明帝令，调三十六龙卫出关。”
<br/>　　此时，边城的马匪也是乱做一团。鬼见愁是十三家马匪之首，因他变态的剑法和变态的性格，将十三家马匪统治在一起。可现在鬼见愁死了，而且死得还稀里糊涂。他这一死，群龙无首，诸家马匪就要回到原先各自为战的情形。
<br/>　　在一番争吵过后，大家商量出结果：谁若杀了程大雷，抢来公主，便是十三家马贼之首。
<br/>　　于是铺天盖地的马匪散开，开始在草原上寻找程大雷的踪影。
<br/>　　于此同时，关内的山贼、强盗、土匪等等江湖豪雄，也都正往关外赶，他们还不知道这里发生的事儿，只想着能从鱼龙卫手里，抢走要和亲的公主。
<br/>　　而从程大雷离开长安城时，崔相已经下令，不能让程大雷活着到达琴川。现在命令改了：不能让程大雷活着离开草原。
<br/>　　命令虽然不同，但意思都是一样的，都需要程大雷去死。
<br/>　　崔相这次出动的人手，包括府中豢养的门客，江湖上的职业杀手。
<br/>　　一时间，在帝国和戎族接壤的这片势力空白区一时间人满为患。包括：帝国特种兵中的佼佼者龙卫，杀人如麻的马贼，职业杀手，顶级剑客，凶悍且不乏高手的山贼，甚至还包括一些要手刃程大雷，为民除害的江湖游侠。
<br/>　　这些人也并非强强联合，大家同心协力追杀程大雷。而是各自为战，谁看见谁也不顺眼，免不得就大战一场。
<br/>　　这里就像是养蛊一样，在放了太多蛊虫的坛子里，只有活到最后的人才是真正的蛊王。
<br/>　　才有可能吃到鸡。
<br/>　　……
<br/>　　废城一战后的夜里，程大雷终于将追兵摆脱。
<br/>　　他们在一个草坡下歇脚，因为害怕被人发现，则没有生火。
<br/>　　毛驴懒散的在一旁吃草，明月在天，空气中有股青草的味道。
<br/>　　程大雷深吸一口气，他一路赶来，见马夺马，路上单是马都累死三匹，如此才算是赶到这里，又从乱军中救出李婉儿。直到现在，程大雷才终于能松口气，卸下一身疲惫。
<br/>　　“嗨。”程大雷摆摆手，道：“想不到我会出现吧。”
<br/>　　看着乱战后力疲倒在草地上的程大雷，李婉儿表情有些无语，她白了程大雷一眼，又轻叹了口气。
<br/>　　“你能不能送我……”
<br/>　　“送你回去嘛。”程大雷摆摆手：“不是问题，一切都包在我身上，我歇口气，然后就把你带回去，我怎么可能让你嫁到戎族。”
<br/>　　“不是送我回去。”明玉公主道：“送我到戎族王城，与戎族王呼延跋成亲。”
<br/>　　本来懒散的程大雷突然坐起，诧异道：“你脑袋吓糊涂了，我听说那呼延跋根本就是个人型走地兽，你愿意嫁给他？”
<br/>　　程大雷声音低下来：“你不是为了救我才答应和亲的么？”
<br/>　　“我愿意。”明玉公主的回答出乎程大雷预料：“既然一定要嫁，顺便救了你有什么不可。”
<br/>　　她也坐在草地上，不顾身上的锦服，看着天空上的繁星缓缓道：
<br/>　　“人生在世上，每个人都有每个人都责任。我从小养尊处优，得到了很多人一辈子都得不到的东西，或许就是为了等这一天，把自己嫁给戎族，成为两族和亲的使者。牺牲我一人，至少能换回两族五年的和平，帝国可以利用这五年休养生息，免的边关黎民百姓流离失所，也避免将士们沙场抛尸。这牺牲是值得的。”
<br/>　　程大雷不知自己面前的是帝国明玉，还是那刁蛮任性的李婉儿，两个身份是一个人，却又是那样的不同。
<br/>　　她长大了……长大不是在时光中的慢吞吞，而只是一瞬间的事情。某一瞬，忽然就明白了，然后折下翅膀，披上盔剑，开始走上战场。
<br/>　　李婉儿说完后，一直没听见程大雷吭声，心里轻轻叹口气。人在江湖，身不由己，于他，怕也是明白了这宿命的无可奈何。
<br/>　　“你这番关于宿命、责任、牺牲的理论，听上去很像一个人？”程大雷试探着问：“你爹忽悠的吧？”
<br/>　　“……”李婉儿。
