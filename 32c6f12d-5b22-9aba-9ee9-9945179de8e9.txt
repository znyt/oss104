　　外门宝库内，萧逸正细致地查看着一件件宝物。
<br/>　　在后山修炼的闲暇之余，萧逸就会来此地。
<br/>　　“四级妖兽精血，100剑点；玄阶高级武技，1500剑点....”
<br/>　　放置宝物的木台上，写明了宝物的等级和用途，以及兑换所需的剑点。
<br/>　　所谓剑点，是裂天剑派的一种奖励点数。
<br/>　　外门弟子，每月可领取30剑点；或者修为突破，也会得到一定的奖励。
<br/>　　另外，剑派也设置了一些任务，可凭此得到剑点。
<br/>　　总的来说，得到剑点不难，但想存到足够的剑点来兑换东西，却非常难。
<br/>　　像一瓶四级妖兽精血，普通的外门弟子，起码要存3个月。
<br/>　　要想得到玄阶高级武技，则起码要4年。
<br/>　　这种方式，有点像猎妖殿。
<br/>　　只不过，猎妖殿纯粹是想驱使天下无数猎妖师为自己效命。
<br/>　　而裂天剑派，则只是一种激励手段，激励弟子们认真刻苦修炼。
<br/>　　只要认真修炼，多突破修为，剑点就会奖励得越多；乃至出外历练，都能得到奖励剑点。
<br/>　　萧逸来此，是因为他现在手头上有许多剑点；第一个考核时，得到的分数，其实就是剑点。
<br/>　　也就是说，他现在足有2535剑点。
<br/>　　萧逸也是来到外门之后，看了剑派的门规之类的东西，才知道这一点。
<br/>　　难怪第一个考核时，顾长风和赵不群等人对分数如此看重。
<br/>　　每个弟子，也就只有在刚刚进入门派参加考核的这个机会，才能通过战斗轻松得到大量剑点。
<br/>　　自然要把握好机会，能抢多少是多少。
<br/>　　正常来说，第一个考核的第一名，一般都能进入内门。
<br/>　　内门宝库，拥有的宝物无论是质量还是数量，都远超外门宝库。
<br/>　　这时，进入内门后，就能凭借从第一个考核中得到的大量剑点在内门宝库兑换一份非常不错的宝物，或功法武技。
<br/>　　这也算是裂天剑派对最出色的弟子的一份馈赠。
<br/>　　只可惜，萧逸现在只是外门弟子身份，不能进入内门宝库。
<br/>　　故此，只能来外门宝库看看了。
<br/>　　当初第一个考核后来这里挑选宝物，由于属于奖励，不需要剑点，限时半天内挑选。
<br/>　　如今需要剑点，则不限时间，外门弟子可随意进入挑选兑换。
<br/>　　剑点来之不易，萧逸自然不会用来兑换武技功法，外门宝库中最高只有玄阶巅峰武技，而且还不多。
<br/>　　他现在暂时不缺这些东西。
<br/>　　他想看看能否找到如三火傀儡球这样的宝物。
<br/>　　剑派内兑换的功法、武技，不得私传，只能自己修炼。
<br/>　　但其他宝物，兵器、天材地宝等，兑换后，则可任凭自己处置。
<br/>　　萧逸想着，半年后回一次萧家，现在给三长老他们准备一些防身宝物也好。
<br/>　　......
<br/>　　一个时辰后，萧逸皱起了眉头，“找了这么久，没一样东西能比得上三火傀儡球。”
<br/>　　终归只是外门宝库，宝物质量远远比不上内门宝库。
<br/>　　这时就看出内门弟子的好处了。
<br/>　　坐拥内门宝库，每月还有60剑点，在其他方面获取剑点也更容易。
<br/>　　若萧逸是内门弟子的话，凭他如今拥有的剑点，不出一年就能存够剑点，换取内门宝库中的地阶武技。
<br/>　　“算了。”萧逸叹了口气，转身离去。
<br/>　　他每日以修炼为主，闲暇时用一个时辰来这里走走，就算找不到宝贝，权当开眼界了。
<br/>　　“嗯？”忽然，一道沉稳而沙哑的疑惑声响起。
<br/>　　“萧逸，我已经注意你好几天了，为何总来外门宝库？”一个老者蓦然出现在萧逸身前。
<br/>　　正是易老。
<br/>　　“额。”萧逸一愣，礼貌道，“参见易老，我只是来随便逛逛罢了。”
<br/>　　“易老怎会来此？”萧逸疑惑地问道。
<br/>　　他发现，这易老还真是修为莫测，以自己的实力，就算是顾长空那等洞玄境七重以上的武者都不可能瞒过自己的感觉。
<br/>　　可易老竟注意自己好几天了，而自己却毫无所觉。
<br/>　　易老面露怪异之色，道，“小子，你不知道我是这外门宝库的守库人吗？我不在这里会在哪里？”
<br/>　　“只不过，我不想被人发现时，便没人知道我在这罢了。”
<br/>　　外门宝库中的宝物虽然阶品普遍不高，但数量逾十万之数，自然是要有守护者的。
<br/>　　“额。”萧逸面露尴尬之色，自从来到外门区域后，他几乎所有时间都独自在后山修炼，对这外门中的人事物大多不熟悉。
<br/>　　易老摆摆手，自顾地走到平日看书的桌案前。
<br/>　　边走边说着，“小子，你已入内门，自当认真修炼，别总来外门区域闲逛。再者，外门宝库中，也没什么宝物值得你来看的。”
<br/>　　萧逸微微躬身道，“易老所言极是，不过我资质愚钝，未能进入内门，便只能来这外门宝库了。”
<br/>　　“小子现在就走，不打扰易老了，告辞。”
<br/>　　说罢，萧逸转身离去。
<br/>　　说实话，一想到自己这几天，在这偌大而寂静的外门宝库中，一直被易老注视着，而自己却毫无所觉，着实是一件恐怖的事。
<br/>　　萧逸并不喜这样的感觉，只想快快离去。
<br/>　　“慢着。”易老叫了一声，疑惑道，“小子，你说什么？你没进内门十堂？”
<br/>　　“嗯。”萧逸应答一声，并不想和这性格古怪的易老独处，头也不回地快步走了。
<br/>　　易老看着萧逸快步离去的背影，眉头皱了起来。
<br/>　　“双考核第一，力压其余考核者，竟然没进内门？”
<br/>　　......
<br/>　　数天后，后山上，萧逸停止了修炼。
<br/>　　他明显感觉到，自己先天九重的修为已经稳固了不少；再精修十天左右，就能彻底稳固下来。
<br/>　　“也该着实准备突破洞玄境了。”萧逸自语一声。
<br/>　　不得不说，他的天赋确实恐怖，突破先天九重还没多久，马上就要准备突破洞玄境了。
<br/>　　洞玄境，乃是武道修炼中的一道分水岭。
<br/>　　一旦突破，就如鱼跃龙门，实力、境界、一切一切都会产生翻天覆地的变化。
<br/>　　故此，这一境界，极为重要。萧逸必须做好充分的准备。
<br/>　　“看来要再去一趟外门宝库了。”萧逸皱了皱眉头。
<br/>　　他记得，外门宝库中有一典籍，就是专门讲述突破洞玄境所需准备的。
<br/>　　而且，那典籍所需剑点不贵，仅仅100点。
<br/>　　毕竟那不是功法武技，只是一本境界讲解罢了。
<br/>　　.....
<br/>　　再次进入外门宝库，萧逸没有像以前那么随意，而是小心翼翼起来。
<br/>　　被人暗中注视而不知的感觉，是非常让人不安的。
<br/>　　但，下一秒，他似乎不用担心了。
<br/>　　因为，易老就坐在平日的桌案前，看着书。
<br/>　　“易老。”萧逸不安地行了一礼，道，“小子这次来，想兑换一物。”
<br/>　　“嗯。”易老不置可否地点点头，道，“带我去吧。”
<br/>　　“是。”萧逸应答一声，来到放置那本典籍的木台上。
<br/>　　“洞玄境突破录？”易老看了眼那典籍，皱起了眉头。
<br/>　　“对。”萧逸没有注意易老的脸色，自顾拿出令牌，准备给易老扣除剑点。
<br/>　　“这东西不必换。”易老摇摇头。
<br/>　　“嗯？”萧逸一愣。
<br/>　　易老凝视萧逸，道，“若你能进入内门，这些知识，自会有内门执事给你讲解。”
<br/>　　萧逸苦笑一声，道，“易老，小子上次不是说了，没能进入内堂吗？”
<br/>　　“我知道。”易老沉声道，“我也知道到底怎么回事。”
<br/>　　“正因为知道，所以无需你兑换。”
<br/>　　“额？”萧逸再次一愣。
<br/>　　易老转身，再次往自己的桌案前走去，边走边道，“回去修炼吧，在修炼上遇到不懂的，可随时来这里寻我，我会给你解答。”
<br/>　　萧逸摇摇头，大概明白易老的意思，但却道，“谢易老好意，小子自己修炼就行了。”
<br/>　　“嗯？”这下到易老一愣，停下脚步，回头道，“你不愿意？”
<br/>　　“不是不愿意。”萧逸苦笑道，“小子虽然知道易老并非普通执事，只是，那顾长空非善类，易老收小子为徒，只怕会惹祸上身。”
<br/>　　“呵。”易老忽然蓦地一笑，“谁说我要收你为徒？”
<br/>　　“只是知道你受了不公的待遇，给你一点儿补偿罢了。在修炼上，有不懂的，你可以来问我，当然，来不来随你。除此之外，你我并无任何关系。”
<br/>　　显然，易老这几天询问过那几个内门执事，知道第三个考核时发生了什么，也知道萧逸为何没进内门十堂。
<br/>　　“另外。”易老再次转过身，负手踱着步，淡淡道，“谁跟你说我是执事的？”
<br/>(本章完)
