　　第310章
<br/>　　程大雷面前几乎已排成人墙，人群密密麻麻围着，用手中的东西交换程大雷的货物。
<br/>　　在他这次所携带的三样货物中，盐是最抢手的价值也最高，其次是药材，然后是茶叶。而程大雷也发现，棉布和铁器在戎族同样是抢手货，下次可以多携带一些。只不过，目前的道路并不支持大型货物的运送，如此开辟新的道路，也就成了当务之急。
<br/>　　程大雷愉快的和众人交易，都说戎族残忍贪婪，也不尽然嘛。在程大雷看来，他们豪爽干脆得紧，只要看得上的东西，出手极其大方，绝不讨价还价。
<br/>　　程大雷用手头的货物换了美石美玉若干，骏马十匹，还有一些草原特产的草药。也有人要拿手中的奴隶与程大雷交换，其中并不乏姿色艳丽的女奴，被程大雷笑着婉拒了。当然，如果不是樊梨花一直守在旁边，用不善的眼神盯着程大雷，程大雷可能会笑，但不会拒。
<br/>　　最后所有货物都出手后，还有人不想程大雷离开，要继续与程大雷交换。
<br/>　　“留下吧，留下吧，我们明天还会再来，大家可要带更多东西过来。”程大雷愉悦的同大家挥手，今天只是来摸一摸情况，明天也才是正式开市的日子。
<br/>　　没想到，程大雷的声音一落地，周围人脸上的表情瞬间僵住了。莫鸣米就搞过这么一次，先用小批量的货物把大家的兴趣钓上来，等着大家赶着牛羊去交易时，莫鸣米布下重兵，抢走了大家的货物，把人杀得干干净净。
<br/>　　如此，也在戎族心目中落下帝国人不守信用的印象。
<br/>　　程大雷这话和当年莫鸣米说得一模一样，该不会又想故技重施吧。众人心里难免都警惕起来。
<br/>　　只是满载而归，心情愉悦的程大雷没有注意到大家的眼神。
<br/>　　他赶着马匹，驮着货物离开，与大部队回合。如秦蛮云中龙等人，也都没有想到，几包盐几包茶叶便可以换来这么多东西，程大雷让大家清点一下，把东西都登记下来。
<br/>　　“大当家，你这一路都没有好好睡觉，今夜你就早点休息，我来守夜。”秦蛮道。
<br/>　　程大雷下意识打个哈欠，这一路行来，的确是没能好好休息。听秦蛮一说，还真的有些困了。他又嘱咐秦蛮几句，然后钻进了帐篷。
<br/>　　秦蛮吩咐人布下警戒哨，自己拄着大枪守在篝火旁。月色朦胧，秦蛮并没有注意到，在不远处的草坡下，有一双双眼睛正向这边望着。
<br/>　　“原来这群帝国人就躲在这里。”
<br/>　　“你留下了盯着他们，我回去报告首领。”
<br/>　　“快些，盯着他们的可不止我们。”
<br/>　　夜里的草原，月牙挂在半空，黑暗里不知藏着多少双眼睛。
<br/>　　次日天明，草原上空气很好，程大雷走出帐篷后慵懒的伸了个懒腰，这一夜睡得他神足气满，连日来的疲惫也缓解许多。
<br/>　　程大雷之所以有这么大的精气神，主要也是因为今天就是正式交易的日子。只要把货物都卖出去，大家就可以打道回府，如果这条商路能够维持下去，日后就是源源不断的利润。而这些收益，就可以支撑起山寨的前期发展。
<br/>　　“出发！”
<br/>　　程大雷骑在马上，身背大斧，腰悬宝剑，意气风发的挥下手。
<br/>　　队伍出发前往红树林，等到的时候，程大雷发现，今天这里聚集的人更多。三五成群，足有七八百人，所携带的各种货物都有，这已经算得上是一个比较繁荣的市场。
<br/>　　更多人前来交易，意味着程大雷可以获得更多的东西，他忙吩咐手下弟兄。
<br/>　　“快，快，把我们的货物都摆上。”
<br/>　　程大雷昨天已经来过一次，大家也都知道他的来意，程大雷还没将货物摆开，已经有人凑过来。
<br/>　　正这时候，忽然响起一阵急促的马蹄声，程大雷打眼看去，见烟尘纷纷，一支戎族骑兵向这边奔来。
<br/>　　不需要程大雷吩咐，队伍已经自动进入防御状态。而程大雷猜得不错，这些人果然是奔着自己来的。
<br/>　　为首一人手持马刀，出现后一言不发，直接向队伍冲来。
<br/>　　程大雷握住了他的手腕，借机发力，蹂身而起，直接将对方从马上摔了下来。而他则骑在了马上，程大雷骑马在林子里转了一圈，稳住马势，居高临下的看着这个人。
<br/>　　这一手玩得分外漂亮，就连以骑术扬名的戎族，此刻也在心中暗暗赞了一声。
<br/>　　“喂，干什么的？”
<br/>　　这人还没从地上站起，秦蛮的枪就压在了他肩上，他抬起头以凶恶的眼神盯着程大雷，却没有说话。
<br/>　　其实也没什么好说的，无非是来抢程大雷而已，只不过手上太弱，只一个回合就让程大雷擒了。
<br/>　　唿律律的马鸣，只见四面都来了人，多的上百人，少的几十人，都骑着马携着兵器，很显然不是来交易，而是来抢劫的。
<br/>　　程大雷轻轻叹口气，自己初来乍到，要么是吃个杀威棒，要么就是亮肌肉，让对方知道自己并不好惹，如此，才能安稳的做生意。
<br/>　　还真是个索然无味的道理。
<br/>　　“秦蛮！”程大雷在马上高喝一声。
<br/>　　“在！”
<br/>　　程大雷伸出手去，扫过一支支队伍，被他扫过的队伍，顿觉脊背发寒。程大雷也在选择，太强的未必打的过，太弱的也不能显示己方实力。
<br/>　　最后程大雷确定一只人数在五十人的队伍，伸出的手重重落下。
<br/>　　“与我破军，斩将，杀敌。”
<br/>　　“是！”
<br/>　　队伍在极短时间展开冲锋，秦蛮一马当先，樊梨花紧随其后，云中龙三个此时也不知该怎么办，最后一咬牙，也操着武器冲了过去。
<br/>　　秦蛮大枪抡开，周围人都不能近身，樊梨花的绣绒刀更是鬼神难测，以这二人为突破口，队伍杀了进去。
<br/>　　主要是蛮字队的攻击来得太快，这些人根本没有反应过来，随后便是首领被杀，其他人落荒而逃。
<br/>　　程大雷骑在马上，冷冷看着这一幕，当他目光扫过其他人时，周围人下意识往后退了一步。
<br/>　　“我们此来，只为做生意，如果有人不好做生意……”程大雷道：“就问问我程大雷的斧头。”
<br/>　　
<br/>(本章完)
