　　“叶先……”
<br/>　　唐清山认出叶尘之后，正要上前行礼。
<br/>　　然而就在这时，
<br/>　　脑海之中，忽然响起了叶尘的声音，
<br/>　　“清山兄，我现在是天海大学的学生，不要暴露我的身份！”
<br/>　　唐清山先是一惊，继而冲着叶尘暗暗点了点头，这才缓步走上前，笑道：
<br/>　　“叶兄弟，您怎么在这儿？”
<br/>　　唐清山虽然已经尽量按照叶尘所说的，没有暴露他的身份，可整个人还是不自觉的恭谨起来，连尊称都用上了。
<br/>　　好在很多人，并没有听清楚，唐清山刚刚用的是“您”这个尊称！
<br/>　　众人看到这一幕，顿时再次一片哗然，
<br/>　　“握草！这家伙是谁？竟然能让唐少如此恭敬对待？”
<br/>　　“我没有听错吧？唐少居然称他为叶兄弟！”
<br/>　　“这到底是何方神圣？”
<br/>　　……
<br/>　　很多人并不知道叶尘的身份，开始暗暗揣测起来。
<br/>　　而杨之凡等人，则个个目瞪口呆。
<br/>　　他们怎么也没有想到，自己这位在平日里，不显山不露水的同学，竟然和大名鼎鼎的唐家太子爷认识！
<br/>　　而且看两人的样子，唐清山还明显是处于下位！
<br/>　　别说众人想不明白，就连杨之凡想破了脑袋，也想不出个所以然。
<br/>　　这倒不是杨之凡太笨，实在是他潜意识里，压根就不敢往那个方向去想！
<br/>　　天海三少看到这个场景，同样也全懵了，他们怎么也没有想到，唐清山居然认识这四人当中的一位。
<br/>　　这下事情可就大条了！
<br/>　　这时，
<br/>　　唐清山也看出来端倪来了，不由得扭过头来，冷冷望着江飞龙三人，寒声道：
<br/>　　“你们三个好大的胆子，竟然敢得罪叶、叶兄弟？还不给我跪下磕头道歉！！”
<br/>　　唐清山此话一出口，众人顿时再次哗然。
<br/>　　纵然唐清山的地位，要远高过天海三少，可是这三人，毕竟是三大家族的嫡子！
<br/>　　唐清山一开口，就要让三人下跪磕头道歉，这实在让人觉得有些过了。
<br/>　　江飞龙三人同样脸色大变，
<br/>　　他们三人是何等身份？在天海市向来横行霸道惯了，别说磕头道歉，便是低头认错也不曾有过！
<br/>　　原来，他们寻思着，冲着唐清山的面子，说声对不起，也就完事了。
<br/>　　却万万没有想到，唐清山竟然要他们下跪磕头道歉！
<br/>　　这简直就是奇耻大辱啊！
<br/>　　“唐哥，你这么做，有些太过分了吧？”
<br/>　　“就是！唐哥，我们叫你一声唐哥，可是你也不能这么折辱我们兄弟吗吧？”
<br/>　　“恕难从命！”
<br/>　　三人不假思索，直接一口拒绝。
<br/>　　唐清山顿时两眼一瞪，寒声道：
<br/>　　“蠢货！我是看在你们父亲的面子，所以才给你们活命的机会，你们可不要不知好歹！”
<br/>　　众人听到这话，顿时一片愕然，全都不明白唐清山这话的意思。
<br/>　　他们又哪里知道，当初叶尘一怒之下，先是灭了天南第一大家族武家，此后又灭了莫家、柳家和陆家！
<br/>　　也正因为如此，才有了江家、王家和方家的崛起。
<br/>　　可是如今的三大家族，连当初的武家、莫家都大为不如。
<br/>　　以叶尘如今的地位和实力，想要灭掉他们，简直就像喝凉水一样简单！
<br/>　　因而，唐清山此举，看似实在羞辱他们，其实却是救他们。
<br/>　　真要是惹怒了叶尘，不但他们三个要丢了小命，恐怕连他们背后的家族，也会跟着遭殃！
<br/>　　可惜，江飞龙三人又哪里知道这些，还以为唐清山仗着自己是唐家太子爷的身份，在故意羞辱他们呢！
<br/>　　就在三人敢怒不敢言，不知道该如何是好的时候。
<br/>　　“乌拉~乌拉~”
<br/>　　一道急促的警笛声陡然响起，
<br/>　　继而，
<br/>　　一辆警车从远处开了过来。
<br/>　　再然后，
<br/>　　警车打开，
<br/>　　一名英姿飒爽的女警，从车上走了下来，赫然正是之前，把叶尘给抓起来的江北警局大队长江飞燕！
<br/>　　“姐！你怎么来了？”
<br/>　　江飞龙看到江飞燕，顿时满脸的欢喜的迎了上去。
<br/>　　原来，
<br/>　　这江飞燕，竟然是江飞龙的亲姐姐！
<br/>　　若是在平时，江飞龙最怕见到自己这位当警察的姐姐，每次都会让他当众难堪。
<br/>　　可是这一次，江飞龙却欢喜的不得了，对他来说，简直就是雪中送炭啊！
<br/>　　唐清山纵然再牛，也不敢公然和警务人员作对，最多也就是事后再找他的麻烦，到时候他只要请家里帮忙，想来应该也能解决。
<br/>　　说到底，这里天海，不是云州！
<br/>　　唐家在天海市还没有完全站稳脚跟，他就不信，唐清山敢在这个时候，和江家撕破脸皮。
<br/>　　别看江飞龙平日不务正业，可是身为家族嫡子，很多问题还是看的挺透彻的。
<br/>　　就连唐清山看到这一幕，脸色也微微一变，不过旋即又恢复了一脸淡然。
<br/>　　他好心搭救江飞龙，既然这家伙不知死活，非要得罪叶尘，那他也懒得管了。
<br/>　　江飞燕看到自己的弟弟，顿时眉头一皱，
<br/>　　“我接到报警，说有人在这里聚众斗殴，该不会又是你小子干的吧？”
<br/>　　江飞龙顿时大呼冤枉，连忙低声将情况向江飞燕说了一遍，
<br/>　　“老姐，这一次我真的什么都没做啊！是唐清山，非要逼我向他的一个朋友磕头道歉，老姐你可要为我做主啊！”
<br/>　　江飞燕听完后，顿时秀眉一蹙，俏脸上浮现出一抹怒色，
<br/>　　“这个唐清山，也太过分了！竟然敢如此羞辱你？他那个朋友又是什么来头？”
<br/>　　江飞龙顿时大喜，当即朝叶尘的方向努了努嘴，
<br/>　　“努，就是那小子！我也知道他是什么来头？好像是唐清山的朋友……”
<br/>　　江飞燕顺着弟弟指的方向望去，顿时美眸一瞪，当即快步走了过去。
<br/>　　江飞龙还以为，江飞燕是要为自己出头，脸上已经露出一抹得意的笑容，心中暗道：
<br/>　　“唐清山，今天本少就让你明白，谁才是天海真正的主人！”
<br/>　　他姐姐江飞燕，是天海市有名的霸王花，江北警察局的大队长，唐家就算再厉害，也肯定不敢公然和警察对着干。
<br/>　　江飞龙心里正想着呢，然而接下来的一幕，却让他目瞪口呆，
<br/>　　只见，江飞燕快步走到叶尘跟前，竟然恭恭敬敬的行了一礼，
<br/>　　“叶……叶先生，您怎么在这儿？”
