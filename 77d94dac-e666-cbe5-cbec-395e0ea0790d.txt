　　第583章
<br/>　　从琅琊城到戎族草原，中间隔着上万里。程大雷单人独骑上路，书院已向外放出消息：要用程大雷的人头为夫子偿命。
<br/>　　程大雷一路上过来，颇遇到过几次伏击，虽然没有威胁到他的性命，的确平添了许多麻烦。
<br/>　　等离开徐州境，进入青州后，路上层出不穷的追杀、伏击才少了些。毕竟，书院虽然是天下读书人的圣地，可主要还是覆盖徐州一地。在青州境内，书院的影响要小很多。
<br/>　　但程大雷仍然不敢掉以轻心，他一路避开城池，大多在野外露营，免得惹来不必要的麻烦。
<br/>　　这一日，程大雷来到一处名唤沙家集的地方，打算在这里买些干粮路上吃，如果有可能的话，再买几套干净衣物。
<br/>　　程大雷牵着牛走进镇子，见街上也没有多少人，无非几个闲汉靠在墙角晒太阳。街道两旁也没有什么店铺，无非一家药铺，一家肉铺，一家酒馆，连一间裁缝铺也无有。
<br/>　　连年征战，十室九空，帝国如今又是群雄割据的局面。能过马鞭的男儿，大多被拉着当兵去了，如沙家集这样的镇子，留下的往往是一些老弱妇孺。
<br/>　　家中有田无人耕，屋里老娘无人养，邻家妹妹白了发，再没见过情哥哥……军中老兵一日一夜唱的，说的大概就是这个意思。
<br/>　　程大雷来到镇中的酒馆，把黑牛搁在后面，也不用担心被谁牵走。因为有程大雷这种人在，所以帝国各地的治安都谈不上好，不过，若是谁想偷程大雷的黑牛，才是自讨苦吃。
<br/>　　他嘱咐老板烫一壶酒，再来半斤肉，有馒头包子之类的热一些，程大雷准备带走路上吃。
<br/>　　酒馆里也无什么人，快要入冬的天气，程大雷又是由南往北去，天气是一日比一日冷了。想起这些日子风餐露宿，程大雷也是苦不堪言。
<br/>　　“掌柜，这附近有没有什么大户人家？”程大雷抿着烧酒，忽然问了一句。
<br/>　　“连年打仗，有钱有势的财主都跑到南边去了，还哪有什么大户人家。”掌柜弓着腰，道：“客官怎么想起问这个？”
<br/>　　“哈，没事随便问问。”程大雷又问道：“那附近有没有山贼？”
<br/>　　“唉，怎么能没有呢，白马峰的白当家，黑虎口的刘黑虎……都是附近有名的山贼。”
<br/>　　“那他们来抢你们么？”
<br/>　　“这……倒也不经常过来。”掌柜道：“沙家集这样的穷地方，也没什么好抢的。”
<br/>　　程大雷呵呵笑了一声，低头喝酒吃肉。程大雷之所以如此问，原因很简单：他手头没银子了。
<br/>　　这一路走来，程大雷路过很多地方，他花钱又大手大脚些，遇上没钱过日子的，免不了就随手抛出去些银子。所以得来的银子，一般用不了多久就被花光了。
<br/>　　而他来银子的办法，往往也只有两条路，一条就是找到附近的大户人家，夜半翻墙进去，偷一笔不义之财。当然，至于对方来钱的路究竟义不义，程大雷其实也不是特别关心。
<br/>　　第二条来钱的路，便是方圆百里之内的山贼大盗。只要程大雷登门报名，不管哪家哪寨，一般都是直接给钱。毕竟，大家既不愿意和他动手，也不想和这魔头扯上什么关系。
<br/>　　事实上，从徐州到青州，程大雷遇到的，不仅是和书院有瓜葛想要杀他的。也有东海一地的江洋大盗，慕名而来，要死要活非要跟着程当家混。相比之下，后者比前者更加麻烦。
<br/>　　毕竟，对于要杀他的，程大雷可以还之以剑，一剑杀了了事。对于那些慕名投奔的，简直可以说是纳头就拜，捧得你天上有地上无，程大雷也不好和他们动剑。
<br/>　　一般打晕了事。
<br/>　　“客官，小店的酒还过得去么？”酒馆掌柜问道。
<br/>　　“嗯呐。”程大雷随口应了一声：“再给我烫上一壶酒，装上些热馒头路上吃。”
<br/>　　“早就给您准备好了。”掌柜殷勤的把包袱皮递过来：“客官您慢走。”
<br/>　　程大雷将干粮搭在牛角上，骑上牛继续赶路。心里琢磨着，自己是要去白马峰，还是去黑虎口，要不两个地方都去一趟。
<br/>　　这里已是幽州和青州的交界，等到了幽州，想要逢到打家劫舍人就不容易了。
<br/>　　也不知怎地，出了镇子后，感觉头越来越沉。程大雷看了头顶的日头一眼，眼皮沉得睁不开。
<br/>　　自己该不会染了风寒吧，自古英雄谁怕死，好汉独怕病来磨，这荒郊野外的，要是自己倒在路上，天寒地冻的，怕怎么死了都无人知道。
<br/>　　咚。
<br/>　　这个念头刚起，程大雷就一头跌倒在地上，勉强抬了两下头，终究挡不住力气的流逝，渐渐失去了力气。
<br/>　　……
<br/>　　程大雷也不知过了多久，他才再次恢复意识，身上还是提不起力气，隐隐约约就感觉耳边有人说话。
<br/>　　“小妹，现在杀不杀？”
<br/>　　“把刀磨得快快的，等他醒来再杀，让他死个明白。”
<br/>　　“要是送官的话，可能换一笔不小的银子。”
<br/>　　程大雷迷迷糊糊睁开眼睛，看到自己被绑在一棵大树上，周围是荒林子，面前有一男一女。
<br/>　　“掌柜？”程大雷愣了愣：“你们……”
<br/>　　那女人三十有余年纪，手中一口牛耳尖刀，模样倒是有几分姿色。
<br/>　　“程大雷，没想到吧，今日你会落在我手里。”
<br/>　　“你是……”程大雷一头雾水。
<br/>　　“你这个没良心的，我被你害成这般样子，你竟连我是谁都不记得了。”女人。
<br/>　　如果是徐神机或者刘发财在此地，听到这样的话，定然会以为程大雷和对方之间发生过什么不可描述的事情。
<br/>　　可程大雷委实记不起对方是谁，无可奈何只能用系统查看对方信息，随之他睁大眼睛，脸上露出难以置信的表情。
<br/>　　“李如……”
<br/>　　女人冷笑一声：“哼，你没想到会有今日吧。”
<br/>　　程大雷感觉头有些疼了，他很想拍拍自己的脑袋，无奈身体被浸湿的麻绳牢牢绑着。
<br/>　　“唉，终日打雁，却被雁啄了眼，阴沟里翻船呐。”
<br/>(本章完)
