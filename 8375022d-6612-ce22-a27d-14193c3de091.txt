　　第584章
<br/>　　面前这女子和程大雷的确有些瓜葛，不过却不是男男女女之间的那种瓜葛。
<br/>　　这李如原先是苏樱父亲的小妾，后来由妾室升格为正房。从苏樱那里算起来，大概还算是程大雷的岳母。
<br/>　　不过二人之间的关系绝对谈不上好，当初落叶城的事情过后，李如全家被赶出落叶城，只靠几亩薄田为生。没过多久，戎族就打了过来，落叶城遭遇屠城。有钱有势的，提前得到消息就往南面逃了，逃不了的就窝在地窖山林中。还没有办法的，也就成了戎族刀下亡魂。
<br/>　　李如的女儿们多死在这次大难中，她和她的兄长李大头则侥幸逃到青州沙家集，无权无势的二人经营一家小酒馆糊口。
<br/>　　凑巧今日程大雷来了沙家集。程大雷最近打交道的都是各路诸侯，再如何也是夫子这样等级的人物，脑海里早已不搁李如兄妹。但李如兄妹是绝对忘不了他，只是隔着门帘望了一眼，李如就将程大雷认了出来。
<br/>　　程大雷和李大头是没照过面，他也的确用系统查看过李大头的信息，但对方当真是个酒馆掌柜，于是程大雷也就没将对方当会事。
<br/>　　也是麻痹大意，没提防对方在酒中下毒，毒也不是太有来历的。无非就是一些药畜生的麻药，纵然酒水的口味有些奇怪，但荒村野店程大雷也不会太挑剔。
<br/>　　说来说去，也是时运不济，程大雷看着面前咬牙切齿的李如，一时有些无语。
<br/>　　程大雷叹了口气，目光打量着李如：“你……还好么？”
<br/>　　“你说呢，我有今日，还不全是拜你所赐。”李如一腔心酸涌上来，眼圈已经有些泛红。
<br/>　　“总是有缘千里来相会，无缘对面不相识。没想到你我能在今日遇见。”
<br/>　　“好贼，难得你还记得我，过去的每一天，我可没有一时一刻把你忘了。”
<br/>　　李大头在一旁听着，感觉二人说话越来越不对劲，忙咳嗽了一声。
<br/>　　“好啦，你们赶快把我放了吧。”程大雷道：“也省的麻烦。”
<br/>　　李如也是恍然回过神来，她的确一肚子委屈，在见到程大雷后难免有几分真情流露。
<br/>　　“放了你！你想的什么好事，我今天定要将你千刀万剐，挖心取肺。”李如咬着牙道。
<br/>　　“啧……”
<br/>　　程大雷想将身上的绳子挣开，他一身力气这也算不得什么难事。可他深吸一口气，发生四肢百骸一丝力气也无，最毒妇人心，药下得够狠呐。
<br/>　　难道堂堂的程当家，当真要死在此地，丧命在名不见经传的李如手中。
<br/>　　“呵，你当我真怕了你不成。”程大雷冷笑一声：“本当家不想死，谁又能杀了我，何况你们。”
<br/>　　“狗贼，死到临头，你却还是嘴硬。”李如提着尖刀：“我现在就把你的心肝挖出来，看你的嘴巴还硬不硬。”
<br/>　　程大雷无可奈何的摇摇头，口中一声呼哨。那黑牛不知刚才跑到哪里去了，此刻狂奔着过来，护在程大雷面前。一个李如一个李大头，二人又有怎样的本事，也就是程大雷一时疏忽，着了二人的道，二人想要杀程大雷，连黑牛这一关都过不去。
<br/>　　只不过两下，黑牛就将二人掀翻在地，李如满身泥泞，被搞得十分狼狈。
<br/>　　程大雷看着二人，眼神中难免有些怜悯，道：“好了，你们也不要白费力气。我包袱里还有些银子，你们拿着去南面，那边的日子终归会好过些。”
<br/>　　“好贼！”李如咬着牙骂了一声，一腔委屈化为愤怒：“你以为我真不能杀你么。就算今日杀不了你，来日我就算化作厉鬼，也不会放过你。”
<br/>　　“住了吧。”程大雷忽然喝了一声，冷笑道：“千错万错，难道都是我的错，归根究底，都是这糟糕世道的错，与我何干。”
<br/>　　李如一怔，她当然想指着程大雷的鼻子破口大骂，泼妇骂街，她的确还是蛮擅长的。可程大雷的话，并非没有道理，她从小孤苦，受人白眼嘲笑，戎族杀过来时，着急逃了，可自己的孩子们却死在戎族刀下。
<br/>　　当然是日日夜夜扎心的痛苦，李如有时候也会想，是否是自己的命不好，是程大雷的错，或许是自己的错……但程大雷的话却给了她另外一种解释：
<br/>　　或许并非是程大雷的错，也不是自己的错，根本就是这世道错了。
<br/>　　无论如何，一个无论如何努力都活不下去的世道，总是有些问题的。
<br/>　　“水中浮萍，风中枯叶，连我都只能随波逐流，更何况你们。”程大雷语重心长道：“听我一句劝，早早往南方去吧，能安稳度过余生，已是你们的造化。”
<br/>　　“程大雷！我不必你装好心，今日我必杀你。”李如提着尖刀站了起来。
<br/>　　“呔，以为本当家当真没有办法么。”
<br/>　　程大雷深吸一口气，李如兄妹的注意力都在他身上，只见他忽然扯着嗓子大喊：“救命啊，救命。”
<br/>　　“……”李如。
<br/>　　“……”李大头。
<br/>　　……
<br/>　　李如对程大雷的畏惧顿消，冷声道：“程大雷，你就算喊破嗓子也无用，今天没人会来救你。”
<br/>　　声音刚落地，耳边忽然一阵急促的马蹄响，几个骑着高头大马的人闯进密林。
<br/>　　为首之人一脸络腮胡子，目光坚毅，他看到面前的情况有些发愣，奇怪道：“什么情况，杀人越货么？”
<br/>　　“嗨，还真被你猜对了，瞧见了没有，这两个贼男贼女是绿林中人，就是要抢我啊。”
<br/>　　这突然出现的大汉名叫韩通，乃是青州王手下的大将，也就是因为程大雷到青州的消息传到这里，才带人出来巡逻，想要逢着程大雷。
<br/>　　也是凑巧，他听到救命声闯了进来，恰好碰到这一幕。还真以为是强人劫财，只是程大雷的反应出乎他的预料，哪有落到强人手中还如此活泼的。
<br/>　　“好汉，你别听他胡说八道。”李如把嘴唇都要咬破：‘他就是程大雷。’
<br/>　　“什么。”
<br/>　　韩通一激灵，看看程大雷面前的黑牛，顿时有种喜从天降的感觉。
<br/>(本章完)
