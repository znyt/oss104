　　第308章
<br/>　　樊梨花是玩刀的，绣绒刀抖开，梨花乱绽。让拿刀的手拿针，的确有些强人所难，所以这靴子做的，不只是丑陋这么简单。
<br/>　　可程大雷双手捧着，却舍不得丢开，这毕竟是女儿心一针一针逢的，千针万线，都是沉甸甸的情份。程大雷心里充满感动，当然，樊梨花能把扶着他背心的手拿开，程大雷或许也没有这么感动。
<br/>　　“嗨，你不就是嫌弃我长得丑么，没有关系，我以前就知道你是拿话诓我，以后一了百了，就当什么都没有发生过。”
<br/>　　说着，樊梨花站起身，还伸手拍了拍程大雷的肩膀。
<br/>　　程大雷一怔，这樊梨花到底是审美观偏到什么地步。
<br/>　　“嘿，你嫌弃我丑，你长得也不好看么。”临走前，樊梨花又补充了一句。
<br/>　　嗳，果然是审美观出现偏差，已到了美丑不分的程度。
<br/>　　说罢，樊梨花便回去窝在毯子上睡熟了，两条长腿卷在一起，像只睡熟的鹿。
<br/>　　窈窕淑女，君子好逑，程大雷也不是不好逑。关键是这姑娘太虎了，程大雷倒现在还处于被吓懵状态。而现在程大雷也是把脑袋拎着在裤腰带上干活，差一分差一秒，说不定就人头落地，程大雷倒也没时间折腾风花雪月。
<br/>　　苏樱如今还在江南，屈指算来已快两年不见，程大雷还等着稳定下来，将她接过来。
<br/>　　话说苏樱和樊梨花却是环肥燕瘦，各有风情，一个如腊月雪，绵里藏针，一个如三月梨花，天真烂漫，若说程大雷更喜欢哪个……
<br/>　　他一个都不喜欢。
<br/>　　与这个世界的隔阂始终都在，身在红尘中，目观世间乱象，他却始终是个局外人。
<br/>　　繁星满天，寒露渐重，程大雷捧着斧头坐在悬崖旁，形单影只。
<br/>　　偶尔回头看樊梨花一眼，佳人的睡相倒是赏心悦目，秀色可餐。
<br/>　　忽然，程大雷注意到一只毛茸茸的手抓向樊梨花，而在黑暗中隐着一头庞大的怪物。
<br/>　　“警戒！”
<br/>　　程大雷吼出声来，第一时间将斧头砸了过去，虽然没有砸中，但是却惊的那怪物退后。
<br/>　　随之，程大雷便扑了过来，樊梨花刚醒，却是被这一幕吓了一跳。程大雷将她护在身后，握住斧头，直面那怪物。
<br/>　　此刻，程大雷才看清怪物模样。
<br/>　　两米身躯，浑身长满黑毛，黑暗中一双眼睛闪烁着瘆人的光芒。
<br/>　　这是一头庞大的山猿。
<br/>　　程大雷握着斧头扑了过去，山猿也捉对冲来。这山猿力大无穷，行动敏捷，实在是山中霸主。程大雷倒也不怵，他力气上比不过，但在速度上有优势，连续三斧头，在山猿肩膀上留下血淋淋的伤口，山猿怪叫一声纵远。
<br/>　　嗷！
<br/>　　山中响起一阵阵啸声，从黑暗中露出一双双眼睛，齐声怪叫，模糊也分不清楚有多少头山猿，但众人此刻已被团团围住。
<br/>　　“怎么没人告诉我这山里有猴子？”程大雷气得大叫。
<br/>　　“我们……我们也没有走过这条路啊。”云中龙胆战心惊。
<br/>　　“警戒，警戒！”
<br/>　　程大雷急忙下令防御。在山中，山猿是称王称霸的存在，集合十个人的力气，也杀不死一只山猿，更何况现在的数目至少有几十头。
<br/>　　面对突然到来的猿群，队伍中的人显出不同状态。秦蛮所率领的蛮字队，虽然也害怕，但却依旧保持着阵型，大家用兵器将马群货物围在当中，随时准备战斗。
<br/>　　而云中龙、乔鹤、游九楼的人，却吓得哆哆嗦嗦，双腿不住颤抖。
<br/>　　“大当家，现在怎么办，您快说句话啊！”云中龙。
<br/>　　“慌什么慌！”
<br/>　　程大雷有些恨铁不成钢的意思，怪不得这帮人屡次被欺负，遇事就乱，毫无战斗力可言。
<br/>　　“大家点起火把围在一起，谁也不要轻举妄动。”
<br/>　　野兽都有怕火的天性，山猿也不例外，大家围在一起，手中挥舞着火把，果然，那些猴子不敢靠前，只是嘴里发出声声怪叫。
<br/>　　发觉果然有效，程大雷稍稍缓解了紧张情绪，可山猿围着不撤，他提着这口气松不下来。
<br/>　　嘭！
<br/>　　突然一颗大石头砸来，幸亏被队伍竖起的长矛挡住，避免有弟兄重伤。随之就是纷纷石头雨，围着队伍的山猿发起了进攻。
<br/>　　程大雷也没想到这些猴子这么聪明，一时也有些手忙脚乱。那些山猿见这一幕有效，已开始尝试进攻。
<br/>　　“挡住，挡住！”
<br/>　　程大雷大喝，抡着斧头冲了过去，幸亏程大雷的队伍经过训练，此时都用装货的麻袋阻挡石子。
<br/>　　程大雷和秦蛮用武器驱赶山猿，可眼前的困境不是二人能解决的。最重要的是被围在队伍中心的马群，马群已经出现骚乱的趋势，一旦这些马失惊后坠入悬崖，那么大家就可以直接回去了。
<br/>　　嗖！
<br/>　　忽然一支羽箭飞出，直直的射入一头山猿的眼睛。然后再发箭，再中，再发箭……
<br/>　　“谁呐，这么好的箭法？”程大雷问。
<br/>　　“是樊寨主！”秦蛮。
<br/>　　程大雷回身看去，只见樊梨花张弓搭箭一气呵成，羽箭一旦脱手，无有不中。
<br/>　　这次队伍没有携带太多弓箭，只有樊梨花自己备着，猿群发出一声声惨叫，渐渐退去。
<br/>　　待猿群退去后，程大雷才重重松口气。他让弟兄们清点货物，为伤员处理伤口，因为这场猿群，足有十几人受伤。
<br/>　　也多亏关键时刻樊梨花的箭法，才没有让这次危机酿成大祸。程大雷回想起来，也是心有余悸。
<br/>　　他本想对樊梨花说几句赞美的话，无奈和樊梨花根本没有搭理他，让程大雷搞得自己有些没趣。
<br/>　　收拾好后，天差不多也快亮了，队伍继续出发。一路上小心谨慎，不敢再有任何大意，在三天后，队伍终于走出了琴川群山，进入了草原。
<br/>　　这里是戎族的地界，以戎族和帝国的大仇，谁看见谁，都是拔刀就杀，众人现在不得不小心。
<br/>　　程大雷自己倒是没多少感觉，他对帝国没感情，心理上和戎族也没有仇。反正大家都是做生意的，只要能赚钱就可以。
<br/>　　而至于这次走过的商路，实在引不起程大雷好感，要想持续经商，必须走一条新的路。
