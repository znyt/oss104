　　“覆海斩。”萧逸低喝一声。
<br/>　　如潮剑音涌出。
<br/>　　数千雪地冰狮，悉数死亡。
<br/>　　数千具妖兽尸体，表面上完好无损，实则体内早已化作了肉泥。
<br/>　　若是再看真切些，将能看到它们幽深眼眸中，竟产生了一抹难以抑制的恐惧。
<br/>　　一个滔天巨浪，倒影在它们的眼眸中。
<br/>　　妖兽虽凶性十足，但也会害怕。
<br/>　　下一秒，萧逸大手一挥。
<br/>　　数千冰狮的妖兽内丹，以及妖兽精血，被悉数吸摄于他手中。
<br/>　　光芒一闪，已收入乾坤戒。
<br/>　　虽然五级内丹和精血对他而言已无大用，但也是一笔价值不菲的财富。
<br/>　　“走吧。”萧逸淡淡地说了一声。
<br/>　　两人，再次赶路。
<br/>　　他并不想在这些普通妖兽上浪费时间。
<br/>　　两人虽无法御空飞行，只能徒步赶路。
<br/>　　但好歹皆是地元境强者，赶路速度还是极快的。
<br/>　　只不过，数天后。
<br/>　　萧逸还是那般模样。
<br/>　　而白冰雪，已是满脸狼狈。
<br/>　　数天时间，两人愈发发现极寒之地的危险。
<br/>　　妖兽的数量、密度，相当骇人。
<br/>　　就萧逸自己所闯过的妖兽森林中，任何一地的妖兽数量，都远远比不上这里。
<br/>　　两人赶路的数天时间内，遇到妖兽，并出手战斗的次数，不下百次。
<br/>　　最麻烦的是，这里的妖兽，都是冰属性妖兽。
<br/>　　或者是环境所致，大多一身白色皮毛。
<br/>　　让得这些妖兽，可以极其轻松地藏身于风雪中，雪地下，忽然偷袭。
<br/>　　加之这里的妖兽并不弱，两人遇到的，最弱都在四级巅峰实力。
<br/>　　想也知道，极寒之地这般寒冷之下。
<br/>　　等闲妖兽，根本活不了。能在这里生活的妖兽，岂会弱。
<br/>　　另外一点，极寒之地，果真无比辽阔。
<br/>　　里头的地形环境，远非雪地那么简单。
<br/>　　冰川河流、银装素裹的雪林、时常爆发暴风雪的冰雪山脉等等。
<br/>　　不同的地方，隐藏着不同的妖兽，袭击手段也大为不同。
<br/>　　加上高空中的极寒罡风。
<br/>　　这一切，构成了这个危机四伏的极寒之地。
<br/>　　就连萧逸初初踏入，也吃了些小亏。
<br/>　　当然，若他自己一人，倒是无所谓。
<br/>　　反倒是白冰雪，已吃了不少大亏。
<br/>　　“萧逸，想到办法没？”白冰雪往日的清冷，变为了疲惫。
<br/>　　短短数天的经历，让她苦不堪言。
<br/>　　同时，她有些惊讶地看着萧逸。
<br/>　　短短数天，萧逸所展现的警惕性、敏锐度、实战经验，几乎强到一个发指的地步。
<br/>　　若非萧逸在一旁照拂，她现在绝对已经受伤，而不是狼狈这么简单。
<br/>　　她甚至相信，若只有萧逸自己一人，哪怕萧逸不是个实力强大的武者，而仅仅是普通人。
<br/>　　都足以安然无恙地在这极寒之地上肆意行走。
<br/>　　此时，萧逸点了点头，道，“有办法了。”
<br/>　　事实上，白冰雪的修为并不弱。
<br/>　　起码这一路遇到的危险，对她而言并不能构成太严重的威胁。
<br/>　　毕竟是地元七重修为，加之每日受剑姬前辈指导武道修炼，战力自是惊人。
<br/>　　只是她实战经验太弱，这才屡屡吃亏。
<br/>　　而萧逸，其实早有办法。
<br/>　　只不过，毕竟受了裂天剑姬不少恩惠，这才特意磨砺白冰雪一番。
<br/>　　此时，两人身处冰川地域。
<br/>　　交谈着，却没注意到，厚厚的冰层之下，一巨大黑影，渐渐靠近。
<br/>　　忽然，轰隆一声巨响。
<br/>　　整个冰层地面，震动不堪。
<br/>　　萧逸眉头一皱，下一秒，低喝一声，“不好。”
<br/>　　连忙一把抱起白冰雪，直直跃起。
<br/>　　几乎是他脚步离开地面的瞬间。
<br/>　　一庞然大物破冰而出，激起百丈冰屑。
<br/>　　“是冰鲸。”白冰雪脸色大变。
<br/>　　冰鲸，七级妖兽，身躯庞大无比。
<br/>　　几乎是这极寒之地的霸主妖兽之一了。
<br/>　　那张巨口，瞬间就能吞下一座冰山。
<br/>　　此刻，那血盆大口，正直直朝萧逸二人吞来。
<br/>　　“裂天斩。”萧逸一道剑气打出。
<br/>　　嗤…
<br/>　　恐怖的剑气，打在冰鲸身上，沿着它的身躯，瞬间将其分尸。
<br/>　　嘭的一声。
<br/>　　失去生机的冰鲸尸体，直直从半空落下，掉下冰川，激起百丈冰浪。
<br/>　　萧逸大手一挥，凌空吸摄了妖兽内丹与精血。
<br/>　　这里的妖兽，麻烦就麻烦在这里。
<br/>　　自身气息完全融入在这冰天雪地中，根本无法感知。
<br/>　　萧逸也只能在它们刚刚露面时，立刻反应过来，或躲避，或反击。
<br/>　　当然，冰鲸这个级别的妖兽，还是比较少见的。
<br/>　　两人遇上，只能说运气背。
<br/>　　“嗯？”这时白冰雪反应了过来。
<br/>　　“萧逸，你能抵挡这些极寒罡风了？”
<br/>　　两人，此时正御空站立于高空中。
<br/>　　萧逸点点头，将白冰雪放下。
<br/>　　迎面而来的极寒罡风，根本无法伤到两人，悉数在萧逸数米之外自动隔绝开。
<br/>　　“这是风雪剑主前辈的操控风雪之法。”萧逸解释道。
<br/>　　风雪剑主，走的是轻柔剑道，以及风雪剑道。
<br/>　　他对风雪一道的理解，极其深奥；对风雪的驾驭，更是极为巧妙。
<br/>　　那御风之能，控雪手段，让萧逸为之惊叹。
<br/>　　之前虽说已经参悟了他的武道知识。
<br/>　　但参悟是一回事，真正掌控，化为己用，是另一回事。
<br/>　　历代剑主的剑道，萧逸以往都只是用来借鉴，提升自身剑道水平。
<br/>　　前几日，才边赶路，边彻底掌控了风雪剑主的手段。
<br/>　　借助这些御风控雪的手段，萧逸以消耗极其微小的元力，就能轻松御空飞行。
<br/>　　且飞行速度丝毫不受影响。
<br/>　　御空飞行下，两人的赶路速度大增。
<br/>　　十天后，彻底来到不夜城。
<br/>　　不夜城，四面有巨大城墙包围。
<br/>　　远远看去，在这漫漫无际的冰雪世界中，像极了一个小黑点。
<br/>　　但当真正来到城墙下，才发现，这座庞然大物，巨大到何等程度。
<br/>　　论占地面积，怕是有炎武王国王都一半左右。
<br/>　　也就是说，足有百城范围。
<br/>　　能在如此一个不毛之地，建立起这样一座冰雪之城，相当了不起。
<br/>　　远方，千年不消的极光，笼罩于不夜城上空。
<br/>　　让得整座不夜城，完全映衬于一片流光溢彩之中，甚是美丽。
<br/>　　这就是不夜城，——横亘于冰雪天地中的不夜要塞。
<br/>　　……
<br/>　　同一时间，极寒之地某处。
<br/>　　一剑眉星目的年轻人，一剑将一头冰鲸分尸后，缓缓地收回了剑。
<br/>　　剑回，星光涌动。
<br/>　　“出来历练近半年了，也不知宗门如何了，现在的我，距离墨寒师兄还有很大差距啊。”
<br/>　　年轻人摇了摇头，“冰缘大会就要开始了，得赶快前往不夜城才是。”
<br/>　　说罢，年轻人御空而起，身旁，无数星影剑光涌动，挡下了高空中的极寒罡风。
<br/>　　虽有些吃力，但终究还是勉强抵挡了下来。
<br/>　　另一边，同样是极寒之地某处。
<br/>　　两道浑身嗜血的身影，踩着地面数之不尽的妖兽尸体，骇人至极。
<br/>　　“喂，叶铭，要不是看在当年你剑派十一长老指导过我的份上，老子才不管你。”
<br/>　　“半年时间，我带着你历练，你修为屡屡突破。”
<br/>　　“也该报答我一番，告诉我紫炎的下落了吧。”
<br/>　　叶铭苦笑一声，道，“无忧兄，你硬把我捉来这极寒之地，已经近半年了。”
<br/>　　“我也说过无数次，我真不知道易兄的下落。”
<br/>　　“你…”钟无忧咬了咬牙，道，“哼，冰缘大会马上就开始了。”
<br/>　　“此等盛事，各方天骄齐聚，我就不信那家伙不去。”
<br/>　　叶铭说道，“无忧兄可能要失望了，据我所知，此次冰缘大会，猎妖殿人员一律不参与。”
<br/>　　钟无忧皱了皱眉头，“哼，就是紫炎不去。”
<br/>　　“萧逸那家伙也会去的。”
<br/>　　“算了，那本公子就先打败他吧。”
