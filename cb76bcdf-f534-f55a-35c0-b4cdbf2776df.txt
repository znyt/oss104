　　“哈哈哈！真是笑死我了！冥沧荨，你这是从哪找来的这么一个傻子？竟然敢说大名鼎鼎的韩月公子是废物！哈哈哈！”
<br/>　　雷洛风顿时又对冥沧荨，开始肆无忌惮的嘲讽起来。
<br/>　　说出这话的同时，雷洛风又在叶尘身上上下打量了一番，脸上的轻蔑之色，顿时越发浓郁，
<br/>　　“区区元婴四重的修为而已，你连本公子都不如，也敢口出如此狂言？真是可笑啊！”
<br/>　　叶尘听到这话，脚下忽然一顿，扭头瞥了一眼雷洛风，嘴角微微一扬，露出一抹玩味的笑容，
<br/>　　“你刚刚说，我不如你？”
<br/>　　雷洛风顿时乐了，脸上浮现出一抹傲然之色，
<br/>　　“怎么？本公子有说错吗？你不过只有元婴境四重修为，而本公子可是元婴五重境界！其实以你的年纪，能够达到这个修为，也算是不错了，可惜啊，你根本不知道，什么才是真正的天才……”
<br/>　　雷洛风絮絮叨叨说个没完，叶尘已经懒得继续听下去，直接随手一挥，
<br/>　　“你给我，跪下！！”
<br/>　　“噗通！”
<br/>　　随着叶尘声音落下，雷洛风顿时感觉到自己，犹如被一座巨山砸中，紧跟着两腿一软，便应声跪倒在了地上，一双膝盖深深陷入地下的岩石之中！
<br/>　　“啊！！”
<br/>　　雷洛风懵了片刻之后，这才感觉到一股钻心般的巨大疼痛，自他的双腿上传来，不由得发出一道凄厉的惨叫声。
<br/>　　其余众人，一时间也全都懵了，他们根本没有想到，这个来历不明的年轻人，会突然对雷洛风出手，更加没有想到，他一出手，竟然就将雷洛风压倒在地，后者似乎毫无任何还手之力！
<br/>　　这家伙，真的只有元婴境四重吗？
<br/>　　就连那五位化神天君，也全都为之一愣，等到众人反应过来，叶尘已经缓步走进了山谷之中。
<br/>　　“少爷！”
<br/>　　雷家众人反应过来之后，当即上前想要将雷洛风从地上扶起。
<br/>　　雷洛风此时早已经气得七窍生烟，捂着自己的双腿，两眼死死的盯着叶尘，怒吼起来，
<br/>　　“杀了他！你们都还愣着干什么？都给老子上！本公子要让他死！！”
<br/>　　“是！”
<br/>　　就在雷家的手下，准备冲上去的时候，却被雷家的那名化神强者一把拦住，
<br/>　　“不用了！这小子敢进入山谷，必然会激怒韩月公子，根本不需要我们出手，他已经是一个死人了！”
<br/>　　众人听到这话，纷纷点头，
<br/>　　纵然之前这个神秘年轻人的表现，有些出乎他们的意料之外，可那又如何！
<br/>　　山谷里的那位，可是八大仙门中的天才人物，就凭他一个区区元婴境的小子，根本不可能活着走出来。
<br/>　　“可惜了！不能亲手宰了那小子！”
<br/>　　雷洛风被下人扶起来之后，依旧一脸恨恨，不由得将怒气又转移到了冥沧荨的身上，
<br/>　　“冥沧荨！这小子是你们冥家的人吧？他竟然敢打伤本公子，此事本公子不会就这么算了的！”
<br/>　　冥沧荨默然不语，美眸死死盯着叶尘的身影，在山谷中渐渐消失，俏脸上顿时浮现出浓浓的担忧之色，
<br/>　　“叶大哥，你可千万不要有事啊！！”
<br/>　　如果可以的话，她一定会拦着叶尘，不让他去触那位韩月公子的霉头，可惜她根本拦不住。
<br/>　　而且以她的修为，即便跟着一起进去，也帮不上什么忙，反而只会添乱。
<br/>　　……
<br/>　　叶尘踏入山谷之后，顿时感觉到那股毁灭气息，越发的强烈。
<br/>　　循着这道气息，叶尘穿过几道崎岖的山路，前方陡然出现了一个巨大的深渊，而且深渊之中，时不时的还喷出，一道道炙热无比的火焰，
<br/>　　“看来灭世魔刀，应该就在这深渊下面了！嗯？下面似乎还有几道颇为不弱的气息，估计应该就是擎苍宗的那位韩月公子了吧？”
<br/>　　一想到擎苍宗，叶尘的脸上，顿时浮现出一抹浓浓的寒意，
<br/>　　前世，八大仙门之中，若说哪个宗门，和叶尘的恩怨最深，肯定非擎苍宗莫属！
<br/>　　记得擎苍宗之中，有一个叫做绝无敌的天才人物，此人觊觎曦月绝世美貌和逆天资质，便以宗门之势，强逼曦月所在的落霞宗，将曦月许配给他。
<br/>　　落霞宗畏于擎苍宗的强大势力，哪里敢反抗，只好答应了这门婚事。
<br/>　　叶尘听说此事之后，直接冲入落霞宗，将曦月带走，两人从此踏入了逃亡之路。
<br/>　　然而，让叶尘没有想到的是，这个绝无敌得知此事之后，迁怒于落霞宗，竟然带入将整个落霞宗，屠杀殆尽！
<br/>　　得知师门被灭的消息之后，曦月痛不欲生，觉得自己愧对师门，一度要和叶尘划清界限。
<br/>　　一怒之下的叶尘，便立誓要屠尽整个擎苍宗！
<br/>　　自那之后，叶尘一边勤修苦练，一边四处猎杀擎苍宗弟子，甚至可以说无所不用其极，短短数年之间，就屠杀了几十万擎苍宗门人！
<br/>　　最终，擎苍宗派出了宗门八大金刚护法，三十六天罡，七十二地煞，合力围剿抓捕叶尘。

<br/>　　也是自那一战之后，叶尘一战成名，跻身修真界超一流强者之列。

<br/>　　再后来，叶尘和八大仙门的恩怨越来越深，甚至到了不死不休的地步！
<br/>　　不过，也多亏了八大仙门的追杀，竟然让叶尘在不停的厮杀之中，战斗力越来越强！
<br/>　　甚至到了最后，即便是八大仙门至尊级的强者出手，也奈何不了他。
<br/>　　但，到了那个时候，叶尘虽然已经屹立在修真界最巅峰的位置，也厌倦了打打杀杀的日子，便带着曦月离开了中央大星域。
<br/>　　一直到叶尘踏入渡劫期，将要渡劫之时，八大仙门不知道从哪里得知了消息，而且还查到了叶尘渡劫的地点，在他渡劫之时，八大仙门的强者忽然杀出，给了他致命一击……
