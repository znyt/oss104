　　第103章
<br/>　　陆哼正带人打扫战场。
<br/>　　乡勇军方面的伤亡是吓人的，轻重伤不计其数，当场死亡者超过五十人。但是，就算付出如此惨痛的代价，今天依旧是一场了不起的胜利。
<br/>　　留下戎族十三条人命，其中还包括戎族尊贵的小王子。在黑石城与戎族连年不断的战斗中，从未获得过如此惊人的胜利。这份战绩报上去，能为陆哼换来不少军功。甚至与说报到京城，大武陛下还会亲口赞他『帝国猛将』之类的话。
<br/>　　呼嘞嘞！
<br/>　　纷乱的马蹄声，陆哼向北望去，只见北方黑压压一片，看数量约有一百余众。他们穿着麻袍，手中挥舞着短刀，口中发出『呼嘞嘞』怪叫。
<br/>　　“戎族，是戎族！”
<br/>　　陆哼大惊失色，早就该想到，呼尔勒不会带十几个人来劫掠黑石城，他们只是前哨，后面还有更多的戎族。
<br/>　　额头潸潸冷汗落下，亏得程大雷杀了呼尔勒，否则若真让呼尔勒的计策得逞，黑石城必有一场大祸。
<br/>　　“退，撤退！”
<br/>　　陆哼在马上大喊，人荒马乱，这些人并非训练有素的正规兵，只不过一群由才子，猎户，武夫，家丁组成的乌合之众。
<br/>　　他们不懂得如何有序撤退。
<br/>　　双方之间的距离被迅速缩短，相距约百步时，戎族已开始在马上射箭，随着纷纷箭雨，
<br/>　　持弯刀带着狂叫冲入队伍中。
<br/>　　陆哼率领的乡勇军，不堪一击。
<br/>　　最后，陆哼只带着三十余骑逃回黑石城，就连他拼命想要保住的呼尔勒首级，也只能留给对方。
<br/>　　这百人数的戎族队伍，在杀戮一番后，带着呼尔勒的尸首扬长而去。
<br/>　　……
<br/>　　“姓高的，你们想做什么！”
<br/>　　“白当家，何必明知故问，把东西留下，饶你们不死。”
<br/>　　“混蛋，你休想！”
<br/>　　“得了，能做山贼谁不混蛋，好像你是什么好人似的。”
<br/>　　山道之上，两只队伍对峙，一方势单力孤已经被逼到一棵大树下。据树而守，所有人围成一团，另一方一时也攻不下。
<br/>　　“白当家，看在绿林道的情分上，我不想把事情做绝，你今天真逼着我杀人么。”
<br/>　　“你做得已经够绝了。”
<br/>　　“哼，如此也就别怪我姓高的不是人。”
<br/>　　正在这个时候，一队人沿着山道缓缓而来，骑马经过这里。
<br/>　　“喂，你们干嘛呢？”
<br/>　　“飞虎寨在此做事，不想死……”
<br/>　　高飞豹回过头来，看清说话人的模样后，表情十分崩溃。
<br/>　　“怎么是你？”
<br/>　　“怎么不能是我？”程大雷摸摸鼻子：“我要回家啊。”
<br/>　　小白狼大喊：“程当家救我！”
<br/>　　高飞豹看看程大雷，突然一跺脚：“不打了，不打了。”
<br/>　　三家势力又聚到一块，小白狼冲程大雷道：“多谢程当家出手，大恩大德容我改日再报。”
<br/>　　问清楚是怎么会事，飞虎寨和杏花岭一道去黑石城打劫，双方各有目标。结果杏花岭抢劫成功，飞虎寨却一直没找到下手的机会。后来城里城外风声紧了，高飞豹再无法下手，只好带着人无功而返。
<br/>　　双方一起回山寨，这次无功而返，高飞豹心情本来就很崩溃，却不料小白狼的手下还不停炫耀这次抢劫的战利品。
<br/>　　类似于『这块布我回去可以做个袍子』『这枚耳环真好看』之类的话。
<br/>　　高飞豹一咬牙一跺脚，特么，我干脆劫了你们。
<br/>　　听明白怎么会事后，程大雷道：“这就是你们不对了，高二爷心里本来就不痛快，何苦你再说风凉话。”
<br/>　　“姓程的，不用你装好人。”高飞豹道：“你么这次成果怎样？”
<br/>　　“收获满满啊。”
<br/>　　高飞豹目光转了一圈，心想：装什么装，还不是和我们一样，什么都没抢到。
<br/>　　他却并不知道，程大雷这次最大的收获就是赵子龙，高仿版赵云呐！至于财物，本就不是程大雷这次的目的。不过，程大雷这次行动，总得来说也不是很顺利，真要某些事传出去，还不被青牛山的山贼笑掉大牙。
<br/>　　还是不说的好。
<br/>　　三方势力一起赶路，程大雷隔着两家之间。小白狼还因为刚才的事情不快，一路上没有再与高飞豹答话。不过，也仅仅是不快而已，山贼么，无非就是你抢我，我抢你，换个时候，小白狼抢了高飞豹也说不定。
<br/>　　高飞豹说得不错：都做了山贼，谁特么不是混蛋！
<br/>　　不过程大雷难免还要奚落高飞豹两句：“高二爷，本是同根生，相煎何太急，大家都是山贼，怎么能向自己人下手呢？”
<br/>　　“姓程的，不必你说风凉话，合着你们两家不用饿死人。”高飞豹阴冷的笑着。
<br/>　　程大雷摸摸鼻子，自己只不过信口说一句，怎么高飞豹的反应这么大，恼了！
<br/>　　……
<br/>　　青牛山，蛤蟆岭。
<br/>　　嗖！
<br/>　　一支羽箭发出破空声，刺穿空气，正射在蛤蟆寨的大门上，箭尾轻轻颤动。
<br/>　　“高当家，咱们向来井水不犯河水，有什么事不好商量，何苦动刀动枪！”黄三元站在门头上大喊。
<br/>　　顺着黄三元的视线往下望，只见蛤蟆岭下黑压压一边，飞虎寨高飞豹，秃头坡熊大熊二，野猪岭一阵风……青牛山五十多家山寨，今天竟都到了。
<br/>　　高飞豹马上喊：“黄总管，莫怪我们把事情做绝，兄弟们也是混口饭吃，你们把大门打开，我们保证不杀人。”
<br/>　　“髙寨主，我们寨主不在，有些事能否等他回来再说。”
<br/>　　废话，我就是知道他不在，他如果在的话，我还能打下蛤蟆寨嘛。
<br/>　　“黄三元，莫要敬酒不吃吃罚酒，数十之内，如果再不开门投降，兄弟们就要打上去了！”
<br/>　　黄三元急的满头大汗，山下的人数足有数千人，其中既有七十老叟，也有黄口幼童，整个青牛山的山贼，不分老幼，竟然倾巢而动。
<br/>　　蛤蟆寨虽有防御设施，但挡不住这许多人。
<br/>　　况程大雷这次行动，把山贼中能打的都带走了，留在山寨的只有关鱼一人而已。
<br/>　　“关兄弟，你看他们如此雄壮，吾等该如何是好？”
<br/>　　关鱼眼睛眯成一道线，睥睨的神色道：“以吾观之，土鸡瓦犬耳。”
<br/>　　黄三元又道：“黑马之上，铜盔铁甲者就是高飞虎。”
<br/>　　“吾观其，如插标卖首耳。”
<br/>　　黄三元一激灵，好熟悉的感觉，该不会又是一个韩玄之吧。
