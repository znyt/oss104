　　第499章
<br/>　　程大雷领着一众人从蛤蟆城中涌出，一路追出来，追出城外五里，总算是追上了将要走的李婉儿。
<br/>　　诸多人将李婉儿围在当中，场面看上去有些吓人。程大雷松了口气，走到李婉儿面前，道：“留下吧，哪儿干不是干呐，现在外面世道很乱的。”
<br/>　　李婉儿抿嘴一笑，道：“这是你的地盘，不是我的，我想出去转转。”
<br/>　　程大雷白她一眼，道：“我的不就是你的么，都是一家人，还分什么你的我的。”
<br/>　　李婉儿笑着摇摇头，没有回应程大雷的玩笑。
<br/>　　程大雷也终于收起玩笑的心思：“怎么要走呢，难道真是为了……传教？”
<br/>　　李婉儿抿起嘴：“李家对不起天下人，身为李家的子孙，总是要去做些事。我不能再在这里养尊处优，想要出去走走，看看天下是什么样，或许也能帮一二个百姓。”
<br/>　　程大雷有些无语，李家人有一个算一个，脑袋都有些拎不清。李婉儿闷闷不乐这么久，最后怎么把自己变成了一个圣人。
<br/>　　“你手不能提，肩不能扛，现在世道的确有些不太平，你一人出去，说不得会遇到怎样的事。况且凭你一个人，又能做得什么事呢？”
<br/>　　李婉儿道：“能做多少便做多少，若真死在荒山野岭，怕也是命数。”
<br/>　　“命数？”程大雷道：“你们万物教还信这个？”
<br/>　　李婉儿避开程大雷的眼神：“我去意已决，你不用再说了。”
<br/>　　“我不许你走。”程大雷板起脸道。
<br/>　　“程大雷，你的好意我懂，然而，我总有我想做的事，你留我在此地，却也只能留我一具行尸走肉而已。”
<br/>　　程大雷默然，他不得不承认，李婉儿的话也并非没有道理。自己把她从草原上带到这里，如李行哉所说，那时帝国明玉公主已经死了，李婉儿可以心无挂碍的活着。但这种话也只是说说而已，一个人没那么容易死的。
<br/>　　国破家亡，说出口只是四个字，但放在一个人身上是怎样的感受，程大雷却无法体会。面对这种经历，再让李婉儿无忧无虑的活在蛤蟆城，几乎也是不可能的事情。
<br/>　　看着李婉儿立在自己面前的样子，虽然是粗布麻衣，但脸上却有了精气神，和之前在山寨时憔悴的样子不同。
<br/>　　或许，自己并不该留她。
<br/>　　“临走前，我原想与你说一声，但也知道你现在的状况。你似乎遇到些麻烦，很严重么？”李婉儿问。
<br/>　　“有些麻烦，但不算太麻烦，总是能解决掉的，我担心的还是你。”程大雷顿了顿，忽然道：“李莫愁。”
<br/>　　李莫愁不知程大雷为何会突然点她的名字，向前走了两步，目光看着程大雷。
<br/>　　程大雷行了一礼，道：“仙子，某拜托你一件事，想你随殿下同去，一路保护殿下的安全。日后但有用得着程某的地方，程某拿命还你。”
<br/>　　李莫愁微微一顿，干脆而痛快的答应：“好。”
<br/>　　李婉儿一怔，道：“你不该……”
<br/>　　程大雷挥手打断了她的话，道：“我许你走，这件事你该答应我，不然，我真不许你走了。”
<br/>　　李婉儿心中明白，程大雷是用自己性命换李莫愁的保护，但此时此刻，她竟也说不出拒绝的话。当然，程大雷也不可能让她拒绝。
<br/>　　她微微点头，算是答应了程大雷，也算是向程大雷告辞。
<br/>　　在诸人的视线中，她和李莫愁并肩离去，二人牵着一头小毛驴，一一步一步缓缓而行，最后身影渐渐消失在地平线里。
<br/>　　程大雷身边立的人已无几，大半人返回蛤蟆城，留下程大雷和李婉儿单独说话。现在也就刘悲几个还陪着程大雷。
<br/>　　“大当家，人已经走了，咱们回去吧。”
<br/>　　“嗯。”程大雷点点头，显得有些失魂落魄，人各有志，不可强求，他也并不知道，自己放李婉儿离开，是对是错。
<br/>　　不过有李莫愁保护，总是出不了大碍，也不知道李婉儿这一走，最后能走出个怎样的前程。
<br/>　　程大雷失魂落魄的转过身来，要随刘悲几人一起回城。
<br/>　　也就是这一瞬间，程大雷全身汗毛倒竖，心脏猛地提起来，觉察到致命的危险。
<br/>　　像是被死亡盯上，死神已经冲着自己脖子挥下了镰刀。
<br/>　　这种感觉很是熟悉，那个要自己命的女人……来了。
<br/>　　刘悲虽然本事不如何，但反应是一等一的快，他立刻觉察到程大雷的异样，沉声问：“大当家，怎样？”
<br/>　　锵啷一声，匹夫剑出鞘，他握剑在身前，警惕的盯着四周，寻找那女人的位置。
<br/>　　“退，退，退。”
<br/>　　程大雷口中连声道，刘悲这些人战斗力不如和，碰上那女人就是死路一条，也唯有自己，可以凭借手中剑与那女人对上两招。
<br/>　　同样是绝世，但程大雷的实力主要在乱军丛中发挥出来，而聂隐娘的职业就是刺杀。
<br/>　　刘悲没有任何犹豫，立刻收拢手下，然后，隐在人群中的聂隐娘也就出现在的大家面前。
<br/>　　她刚才隐在人群之中，距离程大雷并不远，可奇怪的是，刚才无有一人察觉。
<br/>　　当她想隐藏时，宛若遁入清风，可当她绽放杀机时，灿烂若烈日。
<br/>　　程大雷与其目光对视，针尖对麦芒，聂隐娘第一时间扑向程大雷。
<br/>　　“狗贼，去死吧。”
<br/>　　程大雷窝在地牢多久，她便守了多久，而聂隐娘的日子当然也不会好过。守到今天，终于是将程大雷守了出来。
<br/>　　前些日子所受得屈辱，今日要一刃洗净。空气中铛铛声响，都是兵器碰撞的声音，刘悲本还想上前助程大雷一臂之力，但他发现这程度的战斗，自己根本没办法插手。
<br/>　　二人都是以快打快，眼前只有两道残影，刘悲没想到程大雷的本事原来已如此了得，可更可怕的是，聂隐娘的本事更加玄妙，在二人对峙的状态下，程大雷明显被压制。
<br/>　　程大雷刺出了一剑，阿飞快剑第一式直来直往，程大雷例不虚发的一剑。
<br/>　　但今天破例了，因为聂隐娘比他更快。
<br/>(本章完)
