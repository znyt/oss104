　　第7章
<br/>　　程大雷从没觉得林少羽如此可爱，比算无一策的徐神机还要可爱，比可爱多都可爱。
<br/>　　“这小子又来纠缠，干脆杀了他！”徐神机一发狠。
<br/>　　“混蛋！”程大雷冷喝：“林少侠是我蛤蟆寨的恩人，大大的恩人，谁要杀他就是和我过不去。”
<br/>　　徐神机一头雾水，这个人怎么会是蛤蟆寨的恩人，他对我们有什么恩？
<br/>　　“林少侠，饿不饿，要不要先吃个饭，我们山寨的手艺还是很不错的喔？”程大雷眯起眼睛。
<br/>　　“恶贼，休得废话，那个大汉你快快过来，这一次你不骑马，我也不骑马，你不用兵器，我也不用兵器！”林少羽将铁枪扎在地上。
<br/>　　程大雷冲秦蛮道：“秦大哥，万万不可伤了林少侠，温柔些，一定要温柔些。”
<br/>　　“嗯。”
<br/>　　秦蛮闷哼一声，大步向前，林少羽双拳刚刚架起，衣领已经被秦蛮拎起，顺着山坡抛下去。
<br/>　　“哇，啊呀……哇……”
<br/>　　“林少侠，你没事吧，我们在等你，你抓紧时间快点上来……”程大雷冲着山下大喊。
<br/>　　嘟，第二次山寨防守任务完成，奖励山泉一口
<br/>　　嘟，奖励还血丹10枚（普通丹药，强身健体，可以治愈一般伤病）
<br/>　　嘟，奖励山寨技能：齐心（山寨内人口忠诚度增幅）
<br/>　　……
<br/>　　……
<br/>　　嘟，奖励镰刀十柄
<br/>　　嘟，奖励木箱五口
<br/>　　嘟，接到任务『第三次山寨防守』
<br/>　　第二次的奖励明显比第一次水很多，最后镰刀都搞了出来，而且新任务连难度提示都不给，系统，你很皮喔。
<br/>　　一个人影拄着木棍从山下爬上来，他衣服被划破，头上顶着枯草，脸上沾着灰尘。
<br/>　　“林少侠，真的无事吗？”
<br/>　　“你们不放出苏家小姐，我绝不罢休！”林少羽一字一顿。
<br/>　　程大雷无可奈何的挥挥手：“温柔些。”
<br/>　　嘟，完成『第三次山寨防守』任务，奖励……
<br/>　　嘟，接到任务『第四次山寨防守』
<br/>　　嘟，完成……奖励……
<br/>　　嘟，接到……
<br/>　　嘟，完成……
<br/>　　……
<br/>　　……
<br/>　　从早到晚，程大雷足足刷了十次BUG，最后程大雷都有些麻木，奖励什么也不仔细看。当然系统给出的奖励也越来越水，第十次任务干脆只给一口木箱。
<br/>　　程大雷还没时间把奖励都找出来，现在大概是下午三四点左右，程大雷还想趁着天黑再刷几次任务。可林少羽在第十次被赶下山后，一直没有上来，以他的性格，不像轻易放弃的人呐。
<br/>　　“大当家，咱们回去吧，我都饿了。”徐灵儿。
<br/>　　“等着，不能让林少侠失望。”
<br/>　　程大雷一边等，一边翻检脑海中的系统奖励，其中一条特殊的提示吸引他的注意。
<br/>　　嘟，完成十次山寨防守没有造成伤亡，奖励隐藏属性：一个好人。
<br/>　　现在程大雷也摸索出些关于系统的信息。系统任务都是满足一定条件后才会被触发，隐藏任务的触发条件更加苛刻。而有些任务，系统并不会发布任务，只会在满足完成条件后，才会给出完成提示。
<br/>　　其实这次还是被程大雷钻了空子，假若攻打山寨，少则十几人，多则上百人，怎么会没有伤亡。偏偏林少羽就是一个人来的，而且一来来十次，程大雷怎么舍得他死，当宝贝供起来还来不及。
<br/>　　只是不知道自己的隐藏属性有什么作用，程大雷查看着提示信息。
<br/>　　一个好人
<br/>　　『正义的少年喔，在乱世中保持自己的善良是如此珍贵，请铭记，善良的人运气不会太差，你的正义会吸引更多志同道合的少年』
<br/>　　一旦系统开始卖萌，程大雷就知道没好事。这属性……是系统钦定的好人卡么，喂，歧视单身狗有罪好不好。现在程大雷有理由怀疑，这是系统被自己刷了BUG后特意恶心自己。
<br/>　　对于系统属性，要仔细分析，才能知道它的作用。比如这个『吸引更多志同道合少年』，应该是对男性有一定魅力值加成，对女性……没用。
<br/>　　特么，还是好人卡！
<br/>　　程大雷狠狠比出中指，这绝对是系统故意恶心自己。
<br/>　　等程大雷回过神来，发现林少羽还没有上来，以林少羽的性情，该不会如此轻易放弃。他会不是遇到坏人……
<br/>　　“喂，你们在这里等着，我下山看看。”
<br/>　　程大雷将心一横，扛起自己的宣花斧，谁敢伤我的林少侠，我与你们一天二地仇，三江四海恨，誓不罢休！
<br/>　　……
<br/>　　一人一杖一山崖。
<br/>　　林少羽拄着木杖立在悬崖前，头发乱糟糟披散开来，脸上青一块肿一块，身上白袍更是破烂如乞丐装。
<br/>　　望着深不见底的悬崖，林少羽发出一声长叹，眼中长泪滴落。
<br/>　　“枉我林少羽一直以侠客自居，仗枪出世，本想凭手中一口枪，扫平世间腌臜。可今日却一而再再而三在山贼手中受辱，现在马也丢了，枪也丢了，罢罢罢，死了吧，死了吧，二十年后又是一条好汉，我再闯江湖。”
<br/>　　说着，语带悲声，眼中长泪不停，脚步向山崖迈出一步。
<br/>　　“咳咳。”
<br/>　　两道轻咳声响起，林少羽猛地回头，看到程大雷负手立在他身后。林少羽急忙将眼泪抹掉，冷冷问：“蟊贼，你来看我笑话吗，林某就算技不如人，也宁死不受你辱。”
<br/>　　“刚才你的话我都听见了，说得很好。”程大雷道：“有一句话，我觉得说得很好，想和你分享一下。”程大雷看着林少羽的眼睛道：“少年肯为一个理想去死，真正的男人却肯为理想卑微的活着。”
<br/>　　一句话如闷雷在林少羽脑中炸响，炸得他心脏乱跳，面如死灰。
<br/>　　“你觉的这样做很勇敢嘛！”程大雷忽然大声道：“不，这不是勇敢，这是软弱，你就是个懦夫，逃兵。你看看你自己，那还有半点三好学生，少年侠客的样子！”
<br/>　　这些鸡汤准确的击中林少羽，他虽然不懂什么叫『三好学生』，但依旧觉得程大雷每一句都说到自己心坎中，简直就是迷路时的灯塔，成长路上的至理名言呐。
<br/>　　可这些话，却是从一个山贼口中说出。
<br/>　　“哼，你一个山贼懂什么！”
<br/>　　“我知道你看不起我是山贼，可如果有得选，我也想做个好人。”程大雷摊开双手：“其实我曾经也是一个有理想的人。”
<br/>　　“你竟然还有理想？”
<br/>　　“想听听我的理想么？”程大雷笑笑，忽然觉得也许将林少羽招揽入山寨是个不错的选择。
<br/>　　“……我这双手也曾捧过圣贤书，握过青锋剑，也曾想习得文武艺，货与帝王家，也曾说男儿何不带吴钩，收取关山五十州。可我游历四方，所见流民如猪狗，权贵如虎狼，我发愤于圣贤，只想问这世界为何如此，那时我有一个理想，救苍生于水火，解万民在倒悬，我有一个理想，打破旧世界，重建新乾坤，我有一个理想，让人人有其居，耕者有其田……”
<br/>　　林少羽整个人都懵了，这些话都是从程大雷口中说的么，这简直就应该刻在石上，让后人世代铭记呐。恍惚中，他看到程大雷冲自己伸出手，用满怀深情的目光看着自己。
<br/>　　“现在我这双手，只想握住一个同伴的手，林少侠，你愿意与我一同完成这个理想么？”
<br/>　　剧烈的情绪在林少羽胸腔酝酿，在他眼里，程大雷的身影变得无比高大。晕晕乎乎，他的身体忽然向后倒去，身后就是百米悬崖。
<br/>　　程大雷快走几步，一把将林少羽拽住，此时才发现林少羽已经昏迷过去，身体烫的惊人。他身上伤肿无数，皮肤呈现一种骇人的红。
<br/>　　这混蛋，原来是撑着这么严重的伤病。
<br/>　　没有办法，程大雷先将林少羽扛在背上，又抓起自己的大斧，一步一步向山寨走去。
<br/>　　妈的，这么重。
<br/>　　程大雷咬着牙，脊背都被压弯，迈出的每一步都无比艰难。
<br/>　　好吧，好人卡发挥作用了。
<br/>　　
<br/>(本章完)
